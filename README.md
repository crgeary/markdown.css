# markdown.css

Make HTML look a bit like Markdown

## Demo

Check out [the demo](https://markdown-css.netlify.com) for `markdown.css`

## Supports

* [x] Headings 1 to 6
* [x] Bold
* [x] Italics
* [x] Hyperlinks
* [x] Code (inline)
* [x] Code (block with language identifier)
* [x] Lists
* [x] Strikethrough
* [x] Horizontal Rule
* [ ] Quotes
* [x] Checkbox
